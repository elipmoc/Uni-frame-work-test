# フレームワークについて

**ビックバンアーキテクチャ**の使用をサポートできるようなフレームワーク**UniFrameWork**というものを開発中である。
現在の所、独自のScene遷移の実装がメインである。
**UniFrameWork**自体も**ビックバンアーキテクチャ**を元にして作成している。
